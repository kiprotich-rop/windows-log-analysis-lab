# Windows Security Log Analysis Lab

## Objective

Investigate Windows authentication events and identify suspicious login activity using native Windows Event Logs.

## Tools Used

- Windows Event Viewer
- Windows Security Logs
- Kali Linux (Lab Environment)

## Event IDs Investigated

- 4624 Successful Logon
- 4625 Failed Logon
- 4634 Logoff

## Findings

Multiple failed login attempts were observed before a successful login.

The activity was analyzed to determine whether it represented normal user behavior or a potential brute-force attack.
